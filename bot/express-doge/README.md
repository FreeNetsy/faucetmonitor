**how to install and usage:**

**Termux:**
* `pkg install php`
* `pkg install git`
* `git clone https://github.com/B012ED/express-doge.git`
* `cd express-doge`
* `php bot.php`

**Linux:**
* `apt-get install php`
* `apt-get install git`
* `git clone https://github.com/B012ED/express-doge.git`
* `cd express-doge`
* `php bot.php`
